Problem Overview

In functional data analysis, time-warping functions are used to represent phase variability: differences in timing, alignment, or temporal progression across observed curves. However, warping functions are nonlinear under the conventional L² metric, making direct statistical modeling difficult.

This paper addresses that challenge by constructing a principled linear representation for time-warping functions. The proposed framework enables stochastic-process modeling, simulation, resampling, penalized function registration, and functional outlier detection.

Methodological Framework

1. Warping Functions

Time-warping functions represent phase variability in functional observations.

2. Derivative as Density

The derivative of a warping function is treated as a probability density function on the unit interval.

3. CLR Transformation

The centered log-ratio transformation maps the density representation into a linear L² subspace.

4. Hilbert-Space Model

The transformed functions are represented using orthonormal basis functions in a Hilbert space.

5. Stochastic Process

Gaussian and non-Gaussian stochastic processes can be used to generate flexible warping functions.

6. Inverse Mapping

The transformed process is mapped back to the original warping-function space.

Applications in the Paper

Stochastic Simulation

The framework generates warping functions under Gaussian, Laplacian, uniform, and other stochastic coefficient distributions.

fPCA Modeling

Observed warping functions can be transformed, modeled using functional PCA, and resampled in the transformed space.

Penalized Registration

The representation provides a new penalty term for controlling phase variability during function alignment.

Gradient Optimization

A gradient-based algorithm estimates optimal warping functions under the proposed penalized registration framework.

Functional Boxplot

Warping functions are summarized using a boxplot method that captures centrality, variability, and outliers.

COVID-19 Growth Data

The method identifies states with outlying temporal growth patterns in COVID-19 data.

Key Results

The proposed representation provides a unified statistical framework for modeling, generating, aligning, summarizing, and detecting outliers in time-warping functions. It transforms a difficult nonlinear object into a structure where conventional statistical tools such as stochastic processes, fPCA, regression, ANOVA, and functional boxplots can be naturally applied.
